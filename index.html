PORTFOLIO

A few of my favorite Apps

LANLine

LANLine is an app created to help people find others to game with online. It provides an easy way to find others to play or interact.

Game News

LANLine provides in-app news pulled from many different news sources such as GiantBomb, IGN, and more to bring you the latest most up to date in gaming news.

Search for a game

LANLine has access to over 35,000 games. It shows info about the game and has the option to join a chat unique to each game.

Chatting

LANLine offers an easily way to search and find open channels, as well as the option you create your own channels.

Closed Source

Source code for LANLine is unavailable, it is currently in the process of being deployed in the app store. If you would like to see the code, please feel free to reach out to me!

Alien Invasion

Alien Invasion is a game kind of like Space Invaders. You pilot a small aircraft in space and control your movement by tilting the device. in the midst of dodging incoming projectiles, you also fire your own in an effort to stop the aliens before they reach the ground!

Core Motion

Alien Invasion uses Core Motion to detect when the user tilts the screen resuting in the spaceship moving.

Open Source

Code for Alien Invasion can be seen here https://github.com/Smnelson13/SpaceInvaders.

SpriteKit

SpriteKit handles the physics of the game as well as the game scene.
